  • Clásico juego de Blackjack

Desinstalación

Para desinstalar de su Pocket PC: (WM Clásico / WM Profesional)

  • Vaya a Inicio | Configuración | Ficha Sistema | Quitar Programas.
  • Seleccione el nombre del juego o la aplicación panorámic y pulse "Eliminar".
  • Siga las instrucciones en pantalla.

El Panoramic Blackjack de Windows Mobile se ha desinstalado.
Tenga en cuenta, cuando la actualice su software de Panorámic, debe desinstalar cualquier versión anterior antes de instalar la nueva. En la mayoría de los casos, puede instalar una nueva versión de prueba para actualizar a una más completa posterior a menos que la nueva versión sea una actualización comprada.
