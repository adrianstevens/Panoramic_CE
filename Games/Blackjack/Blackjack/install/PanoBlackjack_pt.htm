Como Jogar

O jogo de Blackjack

Panoramic Blackjack Permite que você seja um jogador de Blackjack assim como o seria em muitos casinos. O objectivo do jogo é obter uma mão totalizando o mais próximo possível de 21 sem “Passar por Cima”, bem como marcar mais que o dealer. As normas especificas do Jogo podem variar dependendo do nível de dificuldade seleccionado. Consulte a secção de Opções para mais informações.

O ecrã mostrará o lado do Dealer no topo, com a mão do jogador em baixo. Os botões de controle de jogo aparecem na parte inferior, pode ser pressionada a tecla menu em qualquer momento para abandonar.

Para inicial um novo jogo, seleccione “Novo Jogo” no ecrã de Menu. Antes de começar a jogar, deve primeiro fazer uma aposta, para fazer uma aposta, seleccione ou prima um dos botões de chamada aposta $ 25, $ 50, $ 100, & $ 500. Não é possível apostar mais do que se tem no banco.

Pode-se sempre adicionar a sua aposta à existente, mas deve tocar em “Reset” para apostar um valor mais baixo. Para iniciar uma mão seleccione o botão “Dar”.

Uma vez as cartas sejam dadas, voçê será capaz de Seleccionar, Ficar, Dobrar, Render, Dividir ou Comprar Seguro dependendo das cartas dadas e o nivel de dificuldade escolhido.

Controles

Blackjack pode ser jogado com ou sem caneta (Se o deu dispositivo suportar). Para jogar com caneta, basta tocar os botões no ecrã para activar.

Para jogar sem caneta, tem duas opções, dependendo do seu dispositivo. Pode jogar com o seu controle direccional, cima/baixo/direita/esquerda para mover o cursor no ecrã branco. Uma que o selector se encontre em torno do botão que deseja activar, pressione o centro do controle de 5 botões.

Para dispositivos com teclado numérico (ex: botões 0-9), todos os botões de ecrã são mapeados para os botões físicos. Durante a fase de aposta, os botões farão o seguinte:

1 - Dar
2 - Reset
3 - Apostar $25
4 - Apostar $50
5 - Apostar $100
6 - Apostar $500

Quando uma mão está em jogo os botões farão o seguinte:

1 - Seleccionar
2 - Ficar
3 - Dobrar (Quando Disponível)
4 - Render (Quando Disponível)
5 - Dividir (Quando Disponível)
6 - Seguro (Quando Disponível)

Contagem de Cartas - A Contagem

A ideia por trás de contar cartas no blackjack é que um baralho de cartas com uma elevada proporção de cartões de alta (dez com valor de cartas e Ases) para cartas de baixo é bom para o jogador, enquanto que o reverso (um baralho com uma elevada proporção de cartas de baixa a cartas de alta) é bom para o Dealer. Contagem de cartas Básicas atribui um positivo, negativo ou nulo valor para cada carta. Como cada carta é tratada, a contagem e execução (“A Contagem”) é ajustada pelo valor atribuído a cada carta. uma mão rica em dezenas e ases melhora as chances do jogador, porque os blackjacks (que oferecem um maior pagamento de outras mãos vencedoras) tornam-se mais comuns, o Dealer é mais provável rebentar para uma mão dura, e os duplos-baixos são mais bem sucedidos. * Atenção - A contagem do cartão pode ser considerada batota em alguns casinos os sistemas de contagem que se seguem:

Opções

Para aceder às opções, Seleccione o botão de Opções no ecrã de Menu.

  • Tocar Som: Alterna o som entre Ligado & Desligado
  • Dificuldade: Muda o nível de dificuldade entre Fácil, Médio e Difícil (Veja em baixo), Aprenda a Jogar
  • Contagem de Cartas: Activa esta opção entre Ligado, A pedido & Desligado.
  • Sistema de Contagem De Cartas: Escolher entre 7 sistemas de contagem de cartas, Assistente Ace/Five, KO, Alto-Baixo, Alto-Opc I, Alto-Opc II, Contagem Zen, & Omega II.

Definições de Dificuldade

Fácil:

  • Jogo jogado com um (1) baralho (Baralha Automaticamente)
  • Blackjack paga 2 para 1
  • Ataca o Dealer em soft 17
  • Mantêm sempre visíveis as cartas do Dealer
  • Mostra o valor actual do jogador e as mãos do Dealer
  • Permite a opção de Render
  • Cinco (5) cartas avaliadas como 21

Normal:

  • Jogo jogado com dois (2) baralhos (Baralha Automaticamente)
  • Blackjack paga 3 para 2
  • Ataca o Dealer em soft 17
  • Permite a opção de Render

Difficult:

  • Jogo jogado com quatro (4) baralhos (Baralha Automaticamente)
  • Blackjack paga 3 para 2
  • O Dealer fica em soft 17
  • Não permite a opção de se Render

Aprender a Jogar:

  • Jogo jogado com um (1) baralho (Baralha Automaticamente)
  • Blackjack paga 2 para 1
  • Ataca o Dealer em soft 17
  • Mostra sempre as cartas do Dealer
  • Permite a opção de Render
  • Cinco (5) cartas avaliadas como 21

Uninstalling

Para desinstalar do seu Windows Mobile Pocket PC: (WM Classic/WM Professional)

  • Vá a Iniciar | Definições | Sistema | Remover Programas.
  • Seleccionar no nome do jogo Panoramic ou aplicação e escolher "Remover".
  • Siga as instruções no ecrã.

A Sua aplicação Panoramic Windows Mobile deverá então ser removida.

Nota, quando actualizar o seu Software Panoramic, deve desinstalar as versões antigas antes de instalar uma nova versão. Na maioria dos casos, pode instalar uma nova versão de experimental para actualizar uma versão anterior ou para actualizar uma versão completa mais antiga, a não ser que a nova versão seja uma actualização paga.
