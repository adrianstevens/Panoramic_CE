Désinstallation

Pour désinstaller de votre Pocket PC Windows Mobile (WM Classic / WM Professional)

  • Allez dans Démarrer | Paramètres | onglet Système | Suppression de programmes.
  • Sélectionnez le nom du jeu ou une application panoramique et appuyez sur "Supprimer".
  • Suivez les instructions à l'écran.
Votre Panoramic Windows Mobile application devrait maintenant avoir été désinstallé.

Notez que lorsque vous mettre à jour vos logiciels panoramique, vous devez désinstaller toutes les anciennes versions avant d'installer la nouvelle version. Dans la plupart des cas, vous pouvez installer une nouvelle version d'essai pour mettre à jour une ancienne version complète à moins que la nouvelle version est une mise à niveau payés.
