Overview

Play this enormously popular 52 card solitaire game on your Windows Phone! Since all of the cards are dealt face up, it makes the game of Freecell one of strategy and skill rather then luck. Almost all of the 2 million included game variations can be solved - can you master them all?

  • Contains all 2 million games included in the PC version
  • Four unique game variations (FreeCell, Baker's, Four Colors & KingCell)
  • Options to change number of freecells, enable strict rules or invert the card order
  • Amazing realistic graphics
  • Auto Complete and SmartStack helpers included for efficient gameplay
  • Game state always saved, even when you turn off your phone/device
  • Uses the same game numbers as the Windows desktop version
  • Ability to manually enter game numbers
  • Win/Lose statistics
  • Listing of game numbers known to be unsolvable
  • Five (5) table colors

How To Play

The game of FreeCell

When the game begins, the cards are shuffled into columns and usually only the top cards are moveable. This can be changed by stacking cards in order from high to low, in alternating colors. The object of the game is to stack all the cards, in order by value (Ace to King) and suit, onto the four empty spaces (Foundations) at the top right - just like solitaire. However, this is not so easy as you only have access to certain cards on each stack. There are four empty cells at the top left where you can place cards in order to move cards around more effectively. Only one card may be on each of these cells - no stacks allowed. While stacks can be moved, it depends on how many cells at the top left are free, as this is merely a shortcut to moving them one-by-one from one spot to the other, via the empty cells. For instance, a stack of six cards cannot be moved unless there are additional spaces free where you have cleared columns. You may move any card or stack of cards onto the free cell that is uncovered when you clear a column. Any move can be reversed using the Undo button, even moves that AutoComplete helps you with. However, only the previous move can be undone, so don't play too hastily!

AutoComplete automatically moves Aces to the Foundations for you, and also moves applicable cards that may no longer be useful to you to the Foundations. But be aware that cards that are moved to the Foundations cannot be moved back. You may turn AutoComplete off in the Options menu.

SmartStack is a smart feature that moves stacks for you. When you select a card that is at the top of a stack, and select a destination that is for more cards than one, SmartStack moves the correct cards for you. If you may move either a stack or a single card, SmartStack asks you which you would prefer.

Baker's Game is played much like FreeCell except you can only place cards of the same suit on top of each other in the columns making this game more difficult. Approximately 20% of the deals are solvable.

KingCell has similar rules to FreeCell except it does not require alternate colors when placing cards on to each other in columns. This game is easier than FreeCell. Traditionally it is played with the "Kings only in empty spaces" option on.

Four Colors follows the same rules as Freecell but all 52 cards are dealt into the first four (4) columns and only seven (7) columns are available. The game is more difficult than FreeCell and about 60% of the deals are solvable.

Options

To access the options, select the options button from the Menu screen.

  • Game type: Select a game to play from FreeCell, Baker's Game, KingCell and Four Colors.
  • Table Color: Choose the background color (blue, green, purple, red, and orange)
  • Play Sounds: Toggle the sound on and off
  • Auto Complete: Turn Auto Complete on/off
  • Available FreeCells: Choose 2, 3, or 4 (default) FreeCells in all games
  • Kings only in empty spaces: Strict play where only hi cards can be moved to empty columns
  • Invert card order: Reverses the order of the values of the cards for all game modes

Reducing the number of FreeCells increases the game difficulty. On traditional FreeCell, over 99.9% of the deals are solvable with four (4) FreeCells, 93% are solvable with three (3) and about 78% are solvable with two (2).

Enabling Kings only in empty spaces is the original method of playing FreeCell. Although it increases the difficulty, it has very little effect on which games are solvable.

Invert card order has no effect on the difficulty but makes for an interesting new twist on the game of FreeCell.

Uninstalling

To uninstall from your Windows Mobile device:

  • Go to Start | Settings | System tab | Remove Programs.
  • Select on the name of the Panoramic game or application and tap "Remove".
  • Follow the onscreen instructions.

Your Panoramic Windows Mobile game or application should now have been uninstalled.
