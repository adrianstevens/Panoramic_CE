Présentation

Jouer à ce très populaire jeu de solitaire de 52 cartes sur votre Téléphone Windows! Étant donné que toutes les cartes sont distribuées face visible, il donne au jeu une certaine stratégie et habileté plutôt que de la chance. La quasi-totalité des 2 millions de variations de jeu inclus peuvent être résolus - pouvez-vous toutes les maîtriser?

  • Contient un ensemble des 2 millions de jeux inclus dans la version PC
  • Quatre variantes de jeu unique (FreeCell, Baker, Four Colors & KingCell)
  • Options pour modifier le numéro de freecells, activer des règles strictes ou d'inverser l'ordre de carte
  • Auto finalisation possible et aides incluse pour une jouabilité efficace
  • Liste des numéros de jeu connu pour être insolubles. Six couleurs de Table

Désinstallation

Pour désinstaller de votre Pocket PC Windows Mobile (WM Classic / WM Professional)

  • Allez dans Démarrer | Paramètres | onglet Système | Suppression de programmes.
  • Sélectionnez le nom du jeu ou une application panoramique et appuyez sur "Supprimer".
  • Suivez les instructions à l'écran.

Votre Panoramic Windows Mobile application devrait maintenant avoir été désinstallé.

Notez que lorsque vous mettre à jour vos logiciels panoramique, vous devez désinstaller toutes les anciennes versions avant d'installer la nouvelle version. Dans la plupart des cas, vous pouvez installer une nouvelle version d'essai pour mettre à jour une ancienne version complète à moins que la nouvelle version est une mise à niveau payés.
