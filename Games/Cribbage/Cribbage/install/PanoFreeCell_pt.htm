Visão Geral

Jogue este muito popular jogo solitária de 52 cartas no seu dispositivo Windows Mobile! Uma vez que todas as cartas estão viradas para cima, isto faz o jogo de FreeCell mais uma estratégia de habilidade do que de sorte. Quase todas as 2 milhões de variações podem ser resolvidas – Será que as consegue dominar todas?

  • Contém todos os 2 milhões de jogos incluídos na versão PC
  • Quatro variações de jogo (FreeCell, Padeiro, Quatro Cores & KingCell)
  • Opções para alterar o número de freecells, Habilitar regras estritas ou inverter a ordem da carta
  • Auto Completar e ajudas SmartStack estão incluídas para um jogo eficiente
  • O estado do jogo é sempre guardado quando desliga o telefone.
  • Usa os mesmos numero de jogo da versão desktop do Windows
  • Capacidade de inserir manualmente os números de jogo
  • Estatísticas de Ganhar/Perder
  • Lista de números de jogo conhecidos por serem impossíveis de resolver
  • Cinco (5) cores de mesa

Como Jogar

O Jogo do FreeCell

Quando o jogo começa, as cartas estão baralhadas em colunas e, geralmente apenas as cartas de cima se podem mover. Isso pode ser alterado pelo empilhamento das cartas em ordem de cima para baixo, em cores alternadas. O objective do Jogo é colocar todas as cartas em ordem de valor (Ás ao Rei) e terno, para os quarto espaços vazios (Fundações) no canto superior direito – tal como o Solitário. No entanto, isso não é tão fácil de conseguir, porque só tem acesso a determinadas cartas em cada coluna. Há quarto células vazias no canto superior esquerdo onde se podem colocar as cartas em ordem para mover as cartas em redor de forma mais eficaz. Apenas uma carta pode estar colocada em cada uma dessas células no canto superior está disponível, esta célula é apenas um atalho para poder mover as cartas de um lugar para o outro através das células vazias. Por exemplo uma coluna de seis cartas não podem ser movidas a menos que existam espaços adicionais que tenha livrado as colunas. Pode mover qualquer carta ou coluna de cartas para o freecell que fica descoberto quando limpar a coluna. Qualquer movimento pode ser revertido utilizando o Butão Desfazer, mesmo movimentos de Auto Completar que o ajudam com. No entanto, apenas mover o ultimo pode ser efectuado, por isso convem não jogar com muita pressa!

Auto Completar move automaticamente Ases para as fundações, e também move outros cartões se aplicável que podem já não ser úteis para as Fundações. Mas esteja ciente que as cartas que são movidas para as fundações não podem ser retiradas. O AutoCompletar pode ser Desactivado no menu Opções.

SmartStack é um recurso inteligente que move as colunas para si. Quando seleccionar uma carta que está no topo de uma coluna, e seleccionar um destino que é para as cartas, mais de uma SmartStack move as cartas correctamente para si. Vocês pode mover uma coluna de cartas ou uma simples carta, SmartStack pergunta qual prefere.

Jogo Padeiro é jogado de uma forma muito semelhante ao FreeCell, excepto que só pode colocar as cartas do mesmo naipe em cima umas das outras nas colunas tornando o jogo desta forma aproximadamente 20% dos jogos são resolúveis.

KingCell tem regras semelhantes às do FreeCell, excepto que não existe a necessidade de colocar as cores alternadas nas colunas. Este jogo é mais fácil que o FreeCell. Tradicionalmente é jogado com a opção “Reis apenas nos espaços vazios” ligada. Quatro Cores segue as mesmas regras do FreeCell, mas todas as 52 cartas são distribuídas para as primeiras (4) colunas e apenas (7) colunas estão disponíveis. Este jogo é mais difícil que o FreeCell e cerca de 60% dos jogos são resolúveis.

Opções

Para aceder às opções, seleccione o butão de opções no ecrã Menu.

  • Tipo de Jogo: Seleccione um jogo para jogar desde FreeCell, Jogo Padeiro, KingCell e Quatro Cores.
  • Cor de mesa: Escolha o papel de parede (azul, verde, roxo, vermelho, e laranja)
  • Tocar Sons: Alternar o son Ligado e Desligado
  • Auto Completar: Ligar Auto Completar Ligado/Desligado
  • FreeCells Disponíveis: Escolher 2, 3, ou 4 (defeito) FreeCells em todos os jogos
  • Reis apenas em espaços vazios: Estritamente jogue onde as cartas altas podem ser movidas para as colunas vazias
  • Inverter ordem das cartas: Inverte a ordem das cartas para todos os modos de jogo

Reduzir o numero de FreeCells aumenta a dificuldade. No FreeCell Tradicional mais de 99.9% dos jogos são solucionáveis com três (3) e cerca de 78% são solucionáveis com dois (2)

Actvando Reis Apenas em espaços vazios é o metodo original de jogo FreeCell. Embora ele aumente a dificuldade, tem muito pouco efeito sobre os jogos que são solucionáveis.

Inverter a order das cartas não tem nenhum efeito sobre a dificuldade, mas contribui para uma reviravolta interessante no novo jogo do FreeCell.

Remover

Para remover de um terminal Windows Mobile:

  • Ir a Iniciar | Definições | Sistema | Remover Programas.
  • Seleccionar o nome da aplicação ou jogo Panoramic e seleccionar "Remover".
  • Siga as instruções no ecrâ.

O seu jogo ou aplicação Panoramic Windows deve neste momento encontrar-se removido.
