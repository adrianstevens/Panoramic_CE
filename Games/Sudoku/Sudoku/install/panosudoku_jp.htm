概要

パノラミック　アルティメット数独は、数独、キラー数独、カックロ、KENKENという4つの楽しいロジックベースのバズルゲームを楽しむことができます。数独は9x9のマスを1から9の数字で埋めていくゲームです。カックロは数独の要素にクロスワードパズルを掛け合わせたものです。1から9の数字をマスに埋めていくのがゲームのゴールになります。クロスワードのヒントとなる数字以外のマスに、ヒントの数字の合計と同じようになる数字を入れていきます。キラー数独は、数独とカックロを組み合わせたゲームです。通常は初期の番号は与えられず、9x9グリッドが各リージョンに分かれ、それぞれのマスにはそのリージョンの合計となる数字が表示され数字の重複がないようマスを埋めていきます。ルールは数独を同じです。

  • 数独、キラー数独、カックロ、KENKENという4つの楽しいゲーム
  • 背景色から10種類のスキンから選べます
  • カメラで撮影した写真などをゲームの背景に設定できます
  • 10億を超えるパズルパターン
  • 電話の電源を切ってもゲームの状態は保存されます
  • 数独とキラー数独は5レベルの難易度設定が可能です
  • カックロは3レベルの難易度設定と3グリッドサイズを提供します

アンインストールする

Windows Mobile Pocket PC: (WM Classic/WM Professional) からアンインストールするには

  • [スタート] → [設定] → [システム]タブ → [プログラムの削除] を開きます。
  • Panoramic game か application を選択して "削除" をタップしてください。
  • あとは画面上の指示に従ってアンインストールください。

これであなたの Panoramic Windows Mobile application はアンインストールされます。

注） Panoramic Software を更新する際は、新しいバージョンをインストールする前に古いバージョンを削除する必要があります。ほとんどの場合、新しいバージョンが有償のアップグレード限定でない限り、古い完全版を更新するために新しいお試し版をインストールすることが出来ます。
