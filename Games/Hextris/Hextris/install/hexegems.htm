Overview

Panoramic HexeGems is a highly addictive logic-based puzzle game using four hexagon gems pieces (HexeGems) placed together in blocks. The goal of the game is to manipulate the HexeGem blocks by moving them side to side and rotating them to create a horizontal line without any gaps. Earn as many points as you can before the HexeGems pile up and reach the top of the screen. Gain bonus points by clearing a line that contains an glowing orb. Each complete line that you make clears away that row of gems and gets you closer to moving to the next level. Watch out, advancing levels means increased speed!

How To Play

The game of HexeGems

The goal of the game is to manipulate the HexeGem blocks by moving them side to side and rotating them to create a horizontal line without any gaps. Each time a horizontal line is created the row will disappear and points are awarded. As you advance to new levels the rate at which the HexeGems fall increases. If you are able to eliminate more than one row with a single HexeGem block, bonus points are awarded. Sometimes, a HexeGem block will contain a glowing Orb. Clear a line containing a glowing Orb and receive extra bonus points. Placing two Orbs side by side will result in an explosion, producing a generous amount of bonus points.

There is a "preview shadow" shown where the piece will land if it continues falling at its current position and orientation.

The game is over when the pieces stack up to the top of the screen.

Controls

You can play HexeGems using the 5-way control on your mobile phone or device. Press Left or Right to move the pieces left or right. Press Up to rotate the HexGem block through all orientations and press Down to make the piece fall faster. Extra points are awarded if you press Down as the pieces fall.

You are able to switch the falling HexeGem block with a new HexeGem block that is being held on the side by touching the new block. You can do this at any time.

You may Pause the game by pressing the menu button.

Options

To access the options, select the Options button from the Menu screen.

  • Game: Choose between the four game types
  • Sound Effects Level: Change the volume level for all sound effects
  • Music Level: Change the volume level for all music
  • Start Level: Change the starting level between 1-9. Remember, the higher the level the faster the gameplay!
  • FPS: Toggle show FPS (frames per second)
