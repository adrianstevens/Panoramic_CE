Comment utilise

Vous pouvez utiliser Panoramic Calc Pro Lite pour effectuer des opérations standard pour lequel vous utiliseriez normalement une calculatrice de poche. Simplement, appuyez sur les boutons à l'écran à l'aide de votre doigt ou un stylet (WM standard/Professional), ou utilisez le sélecteur et le clavier (WM standard). Utilisez le bouton de menu en bas à droite pour passer rapidement à votre calculatrice souhaitée. Le bouton de fichier permet de rendre l'utilisation de la copie et collage de fonction ainsi que contient le propos et options d'écrans. Dans le Menu Options, vous pouvez choisir entre un certain nombre de paramètres, notamment l'utilisation de paramètres régionaux (qui détecte les préférences de votre région pour nombre d'affichage - il provoquera numéros longue pour être rompu avec des virgules ou des décimales, le cas échéant), lire des sons, enregistrement sur bande, mode de calcul, notation, enveloppes personnalisées et le montant des chiffres fixe.

Les calculateurs

Standard : Utiles pour la plupart des calculs en cours d'utilisation quotidienne. Inclut des fonctions comme exposant, racine carrée, pourcentage, inverse et les fonctions de la mémoire. La simple mise en page et les grands boutons rend facile à utiliser avec vos doigts.

Devises Européenne : Transport? Utilisez la Calculatrice de devises mondiale d'échanger 14 devises différentes à l'échelle Europe. Sélectionnez les devises utilisant le simplement et de boutons, puis entrez un montant.

Désinstallation

Pour désinstaller de votre Pocket PC Windows Mobile (WM Classic / WM Professional)

  • Allez dans Démarrer | Paramètres | onglet Système | Suppression de programmes.
  • Sélectionnez le nom du jeu ou une application panoramique et appuyez sur "Supprimer".
  • Suivez les instructions à l'écran.

Votre Panoramic Windows Mobile application devrait maintenant avoir été désinstallé.

Notez que lorsque vous mettre à jour vos logiciels panoramique, vous devez désinstaller toutes les anciennes versions avant d'installer la nouvelle version. Dans la plupart des cas, vous pouvez installer une nouvelle version d'essai pour mettre à jour une ancienne version complète à moins que la nouvelle version est une mise à niveau payés.
