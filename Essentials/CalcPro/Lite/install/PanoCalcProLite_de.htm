So Geht’s

Sie können Panoramic Calc Pro Lite verwenden, um eine der Standardfunktionen auszuführen, für die Sie normalerweise einen handheld-Rechner verwenden würde. Einfach mit ihrer Finger oder Stift (WM Classic/Professional) auf dem Bildschirm Tasten drücken Sie, oder verwenden Sie die Auswahl und die Tastatur (WM-Standard). Verwenden Sie die Menüschaltfläche im in der unteren rechten, schnell Ihre gewünschten Rechner wechseln. Der Datei-Schaltfläche können Sie zu verwenden, der Kopieren & Einfügen Funktion sowie die Info enthält und Optionen Bildschirme. Innerhalb der Startoptionen können Sie zwischen einer Reihe von Einstellungen, einschließlich verwenden Ländereinstellungen (die Ihre Region Vorliebe für Anzahl Display - erkennt es wird verursachen lange Zahlen zu mit Kommas oder Dezimalstellen, gegebenenfalls aufgeteilt werden), Wiedergeben von Sounds, Rechner-Modus, Notation, benutzerdefinierte Skins und der Betrag der feste Ziffern.

Der Rechner

Standard: Nützliche für die meisten Berechnungen im alltäglichen Gebrauch. Umfasst Funktionen wie z. B. Exponent, Quadratwurzel, Prozentsatz, Inverse, Rücktaste und Speicher-Funktionen. Die große Schaltflächen und einfaches Layout machen es einfach zu verwenden, mit den Fingern.

Europäische Währungen: Reisen Verwenden Sie den Europa Währungen-Rechner zum Austausch von 14 verschiedene währungen. Wählen Sie die Währungen, die mit der einfach und von Schaltflächen geben Sie dann einen Betrag.

Smartphone-Steuerelemente

Für nicht-Touch-Screen-Geräte mit standard 12 Schaltfläche Telefontasten - erhalten Sie Zugriff auf die grundlegenden Rechner-Funktionen mit nur den 12 Schaltflächen können. Tippen Sie auf, und halten Sie Schaltflächen, um die sekundären Funktionen zuzugreifen. Halten Sie z. B. die '5' gedrückt zu multiplizieren.
Verwenden einfach die entsprechende Schaltfläche für die Eingabe von Zahlen, grundlegende Operatoren usw., für nicht-Touchscreen-Geräte mit Tastaturen. Erweiterte Funktionen entnehmen Sie bitte der Tastatur-Kontextmenü-Liste.

Menüs, denen alle der Menüs in Calc Pro Lite für optimiert wurden beide Touchscreen und nicht-Touch Screen-Geräte. Sie können "ziehen" das Menü, um einen Bildlauf durchführen oder verwenden Sie das direktionale Pad. Drücken Sie oben oder unten, um zu verschieben der Auswahl, oder drücken Sie links und rechts für schnelle Blättern.

Deinstallation

Um es von Ihrem Windows Mobile Pocket PC (Classic/ Professional) zu deinstallieren:

  • Gehen Sie zu Start | Einstellungen | System | Programme entfernen.
  • Wählen Sie den Name des Panoramic Spieles oder der Anwendung aus, die Sie deinstallieren woollen und klicken Sie auch "Entfernen".
  • Folgen Sie den Anweisungen auf Ihrem Display.

Ihre Panoramic Windows Mobile Applikation sollte nun deinstalliert sein.

Anmerkung: Wenn Sie Ihre Panoramic Software Anwendung auf eine neue Version aktualisieren, sollten Sie zuerst die alte Version deinstallieren bevor Sie die neue Version installieren. In den meisten Fällen können Sie zuerst eine neue Trail-Version installieren, bevor Sie die neue Version als bezahltes UpGrade installieren.
