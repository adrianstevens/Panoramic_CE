    How To Use

    You can use Panoramic Scientific Calc to perform any of the standard operations for which you would normally use a handheld scientific calculator.  Simply press the onscreen buttons using your finger or stylus (WM Classic/Professional), or use the selector and keypad (WM Standard). Use the menu button at the bottom right to quickly switch to your desired calculator. The file button allows you to make use of the Copy & Paste features as well as provides access to the About and Options screens. Within the Options Menu you can choose between a number of settings, including Use Regional Settings (which detects your region's preference for number display - it will cause long numbers to be broken up with commas or decimals, where appropriate), Play Sounds, Record Tape, Calculator Mode, Notation, Conversion Buffer, RPN Stack Size, Custom Skins and the amount of Fixed Digits. The toggle button, located in the middle of the bottom menu bar, will take you back to the last calculator used.

    The Calculators

    Scientific: Includes everything from the Standard Calc, plus many additional functions for algebraic and trigonomic calculations. Use the 2ndF (2nd Function) and Hyp (Hyperbolic) buttons to access additional buttons/functions. Add a constant into your equations by hitting the "Const" button and choosing from you Constants Favorite menu. See Constants to see how to set your Constant Favorites. The scientific calculator with also display a custom RPN layout when the RPN calculator mode is selected.

    Base Conversions: Converts between Decimal, Hexadecimal, Octal, and Binary, as well as giving you access to commonly used bitwise operations.

    Graphing: View your trigonometry or algebra equations on a 2D graph. Enter up to six (6) equations and use the graph to discover coordinates like points of intersection.

    Constants: Choose from over 1500 different Mathematics, Physics, Solar System, or Element constants and use them in your work equations. Easily add them to your Favorites menu by hitting the "+Fav" button. The constants favorite menu will hold up to 12 constants of your choosing. The favorites menu is available on the Scientific calculator by pressing the "Const" button.



    Smartphone Controls
    For Non-touch screen devices with standard 12 button phone keys - you'll be able to access basic calculator features using only the 12 buttons. Tap and hold buttons to access the secondary functions. For example, hold down the '5' key to multiply.
    For non-touch screen devices with keyboards, simply using the appropriate button for entering numbers, basic operators, etc. For advanced functions, please refer to the keyboard shortcut list.

    The Display
    For Touch screen devices you can control many of the features available in the options by tapping on the display. Tap the 'M' memory indicator to see the contents, this will also show all ten memory locations when "Adv. Memory" is selected.
    You can also change the DRG state and the notation by tapping on the display. The RPN stack will also be shown by tapping on the answer area when RPN mode is selected.

    Menus All of the menus in Calc Pro have been optimized for both touch screen and non-touch screen devices. You can 'drag' the menu to scroll or use the directional pad. Press Up or Down to move the selector, or press Left and Right for rapid scrolling.

    Options

    Use Regional Settings will format the display results according to your devices regional settings.
    Play Sounds will enable or disable the 'tap' sound when a button is pressed
    Record Tape will create an plain text file will all calculations noted. A new file is created every time Calc Pro is run. The files are stored in the My Documents Folder.
    Adv. Memory enables the advanced memory features. This will swap the memory buttons with advanced buttons on all calculators. Used in combination with the number buttons you'll be able to access 10 memory locations.
    Calculator Mode: Change the calculator mode between Normal, Algebraic, Direct Algebraic, R.P.N and expression
    Notation: Change the output notation between Normal, Scientific, Engineering, Fraction and Fixed
    Conversion Buffer: Change the size of the buffer used for base conversions
    RPN Stack Size: Change the total size of the stack when using the RPN calculator mode
    Skin: Change the appearance of Calc Pro
    Fixed Digits: Change the number of decimal places displayed when using the Fixed notation.

    Scientific Calculator

    The Scientific calculator gives you access to advanced algebraic and trigonomic functions.

    Buttons

    The Scientific Calculator contains all of the buttons on the Standard Calculator as well as:

    Reset - (2ndf of <-) Restore Calc Pro to default settings
    2ndF - Access 2nd function buttons
    Hyp - Access Hyperbolic functions
    const - Constants button - brings up 12 saved buttons (configurable on the constants calculator)
    DRG - Change trigonometry mode between Degrees/Radians/Gradians
    DRG> - Same as DRG but will also convert the onscreen value
    dms - Change decimal value to degrees/minutes/seconds
    dec - Change dms value to a decimal value
    x! - Factorial - can be a decimal value
    sin - Sine
    cos - Cosine
    tan - Tangent
    log - Logarithm base 10
    ln - Natural logarithm (base e)
    pi - Constant Pi
    asin - Arc sine
    acos - Arc cosine
    atan - Arc tangent
    10x - (2ndf of log) 10 raised to the power of
    ex - (2ndf of ln) E raised to the power of
    rand - (2ndf of Pi) Random number between 0 and 1
    x3 - (2ndf of x2) Cube of value
    Y √ - (2ndf of xy) Y root of
    Ans - (2ndf of pi) Recall last answer

    If you have Selected the R.P.N. calculator mode in the options the follow buttons are also shown:

    X<>Y - Switch the last two values in the stack
    Roll - Roll all values in the stack down (bottom to top)
    Drop - Drop all values in the stack
    EEX - Times 10 to the power of
    CHS - Change sign

    Base Conversions

    The Base Conversions Calculator allows you to convert integer number between Binary, Octal, Decimal and Hexadecimal numbers.

    Buttons BIN - Change calculator to binary mode
    OCT - Change calculator to octal mode
    DEC - Change calculator to Decimal mode
    HEX - Change calculator to Hexadecimal mode
    B3 - Change calculator to base 3
    B4 - Change calculator to base 4
    B5 - Change calculator to base 5
    B12 - Change calculator to base 12
    A - Enter number A (Base 12 and HEX only)
    B - Enter number B (Base 12 and HEX only)
    C - Enter number C (HEX only)
    D - Enter number D (HEX only)
    E - Enter number E (HEX only)
    F - Enter number F (HEX only)
    AND - AND binary operator
    NAND - NOT of AND binary operator
    OR - OR binary operator
    NOR - NOT of OR operator
    XOR - Exclusive OR binary operator
    XNOR - NOT of exclusive OR operator
    MOD - Modulus
    |x| - Absolute value

    Uninstalling

    To uninstall from your Windows Mobile Pocket PC: (WM Classic/WM Professional)

    • Go to Start | Settings | System tab | Remove Programs.
    • Select on the name of the Panoramic game or application and tap "Remove".
    • Follow the onscreen instructions.

    To uninstall from your Windows Mobile Smartphone: (WM Standard)

    • Go to Start | Settings | Remove Programs.
    • Select on the name of the Panoramic game or application and tap "Remove".
    • Follow the onscreen instructions.

    Your Panoramic Windows Mobile application should now have been uninstalled.

    Note, when updating your Panoramic Software, you should uninstall any old versions before installing the new version. In most cases, you can install a new trial version to update an older full version unless the new version is a paid upgrade.
