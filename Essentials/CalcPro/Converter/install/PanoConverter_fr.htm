Comment utilise

Simplement, appuyez sur les boutons à l'écran à l'aide de votre doigt ou un stylet (WM standard/Professional), ou utilisez le sélecteur et le clavier (WM standard). Utilisez le bouton de menu en bas à droite pour passer rapidement à votre calculatrice souhaitée. Le bouton de fichier permet de rendre l'utilisation de la copie et collage de fonction ainsi que contient le propos et options d'écrans. Dans le Menu Options, vous pouvez choisir entre un certain nombre de paramètres, notamment l'utilisation de paramètres régionaux (qui détecte les préférences de votre région pour nombre d'affichage - il provoquera numéros longue pour être rompu avec des virgules ou des décimales, le cas échéant), lire des sons, enregistrement sur bande, mode de calcul, et enveloppes personnalisées.

Les calculateurs

Unité de conversion : Convertit 17 unités différentes mesures, telles que, espace, énergie, cuisiner, économie de carburant, vitesse, temps, température et couple. Commencez par sélectionner votre type de conversion avec le bouton de conversion, les unités de sélection avec l'et de boutons. Maintenant entrer une valeur, et il sera instantanément converti. Cliquez sur l'affichage pour basculer l'et d'unités.

Devises Mondiales : transport ? Utilisez la Calculatrice de devises mondiale d'échanger plus 150 devises différentes à l'échelle mondiale. Les devises mondiales, taux de change sont mis à jour depuis le panoramique Software Inc. site web trois fois quotidiennement. Également inclut les acheter et vendre des taux pour la plupart des devises. Sélectionnez les devises utilisant le simplement et de boutons, puis entrez un montant. La monnaie à s'affichera instantanément avec l'acheter et vendre des prix.

Contrôles de Smartphone

Pour les périphériques d'écran non-touch avec touches de téléphone 12 bouton standard - vous pourrez accéder aux fonctions de la Calculatrice de base utilisant uniquement les 12 boutons. Appuyez et maintenez les boutons pour accéder aux fonctions secondaires. Par exemple, maintenez la touche '5' pour se multiplier. Pour les périphériques non-touch écran avec des claviers, simplement en utilisant le bouton approprié pour les numéros, les opérateurs de base, etc. Pour les fonctions avancées, veuillez vous reporter à la liste des raccourcis clavier.

L'affichage. Pour les périphériques d'écran tactile vous pouvez contrôler la plupart des fonctionnalités disponibles dans les options en tapant sur l'écran. Cliquez sur le m ' indicateur de mémoire pour afficher le contenu, ce sera également afficher tous les emplacements mémoire dix lors de la "Mémoire Adv." est sélectionnée. Vous pouvez également modifier l'état DRG et la notation en tapant sur l'écran.

Menus pour que tous les menus dans Converter Pro ont été optimisées deux à écran tactile et non-touch périphériques de l'écran. Vous pouvez 'déplacer' du menu pour accéder ou utiliser le pavé directionnel. Appuyez sur la touche haut ou bas pour déplacer la sélection, ou appuyez sur la touche gauche et droite pour faire défiler rapidement.

Désinstallation

Pour désinstaller de votre Pocket PC Windows Mobile (WM Classic / WM Professional)

  • Allez dans Démarrer | Paramètres | onglet Système | Suppression de programmes.
  • Sélectionnez le nom du jeu ou une application panoramique et appuyez sur "Supprimer".
  • Suivez les instructions à l'écran.

Votre Panoramic Windows Mobile application devrait maintenant avoir été désinstallé.

Notez que lorsque vous mettre à jour vos logiciels panoramique, vous devez désinstaller toutes les anciennes versions avant d'installer la nouvelle version. Dans la plupart des cas, vous pouvez installer une nouvelle version d'essai pour mettre à jour une ancienne version complète à moins que la nouvelle version est une mise à niveau payés.
