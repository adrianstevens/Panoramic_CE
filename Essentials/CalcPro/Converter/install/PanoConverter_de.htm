Einfach mit ihrer Finger oder Stift (WM Classic/Professional) auf dem Bildschirm Tasten drücken Sie, oder verwenden Sie die Auswahl und die Tastatur (WM-Standard). Verwenden Sie die Menüschaltfläche im in der unteren rechten, schnell Ihre gewünschten Rechner wechseln. Der Rechner

Einheit-Konvertierungen: Konvertiert 17 andere Einheit-Messungen wie Area, Energie, Cooking, Fuel Economy, Speed, Zeit, Temperatur und Drehmoment. Starten Sie, indem Sie den Typ Ihres Konvertierung mithilfe der Konvertierungs-Schaltfläche, die wählen Sie Einheiten mit an und von Schaltflächen. Geben Sie jetzt einen Wert und es sofort konvertiert. Tippen Sie auf die Anzeige der zu wechseln und von Einheiten.

Welt Währungen: Reisen Verwenden Sie den World Währungen-Rechner zum Austausch von über 150 + verschiedener Währungen weltweit. Die Welt-Währungen, die Wechselkurse von der Panorama Software Inc. aktualisiert werden-Website dreimal täglich. Auch umfasst kaufen und verkaufen Sie Preise für die meisten Währungen. Wählen Sie die Währungen, die mit der einfach und von Schaltflächen geben Sie dann einen Betrag. Die Währung an sofort angezeigt werden zusammen mit dem kaufen und verkaufen Sie Preise.

Smartphone-Steuerelemente

Für nicht-Touch-Screen-Geräte mit standard 12 Schaltfläche Telefontasten - erhalten Sie Zugriff auf die grundlegenden Rechner-Funktionen mit nur den 12 Schaltflächen können. Tippen Sie auf, und halten Sie Schaltflächen, um die sekundären Funktionen zuzugreifen. Halten Sie z. B. die '5' gedrückt zu multiplizieren.
Verwenden einfach die entsprechende Schaltfläche für die Eingabe von Zahlen, grundlegende Operatoren usw., für nicht-Touchscreen-Geräte mit Tastaturen. Erweiterte Funktionen entnehmen Sie bitte der Tastatur-Kontextmenü-Liste.

Die Anzeige

Für Touch-Screen-Geräte können Sie viele der Funktionen in den Optionen durch Tippen auf der Anzeige steuern. Tippen Sie auf das m ' Speicher Indikator, der den Inhalt, dies wird auch finden Sie unter Anzeigen alle zehn Speicherbereichen, wenn "Advanced Memory" ausgewählt ist. Sie können auch den DRG-Zustand und die Notation durch Tippen auf der Anzeige ändern. Der RPN-Stack wird auch durch Tippen auf den Antwort-Bereich angezeigt werden, wenn RPN-Modus ausgewählt ist.

Menüs, denen alle der Menüs in Converter Pro für optimiert wurden beide Touchscreen und nicht-Touch Screen-Geräte. Deinstallation

Um es von Ihrem Windows Mobile Pocket PC (Classic/ Professional) zu deinstallieren:

  • Gehen Sie zu Start | Einstellungen | System | Programme entfernen.
  • Wählen Sie den Name des Panoramic Spieles oder der Anwendung aus, die Sie deinstallieren woollen und klicken Sie auch "Entfernen".
  • Folgen Sie den Anweisungen auf Ihrem Display.

Ihre Panoramic Windows Mobile Applikation sollte nun deinstalliert sein.

Anmerkung: Wenn Sie Ihre Panoramic Software Anwendung auf eine neue Version aktualisieren, sollten Sie zuerst die alte Version deinstallieren bevor Sie die neue Version installieren. In den meisten Fällen können Sie zuerst eine neue Trail-Version installieren, bevor Sie die neue Version als bezahltes UpGrade installieren.
