Overview

    How To Use

    You can use Panoramic Converter Pro to perform currency or unit conversions.  Simply press the onscreen buttons using your finger or stylus (WM Classic/Professional), or use the selector and keypad (WM Standard). Use the menu button at the bottom right to quickly switch to your desired calculator. The file button allows you to make use of the Copy & Paste feature as well as contains the About and Options screens.

    Unit Conversions: Converts 17 different unit measurements such as, Area, Energy, Cooking, Fuel Economy, Speed, Time, Temperature and Torque. Start by selecting your conversion type with the Conversion button, the select units with the To and From buttons. Now enter a value and it will be instantly converted. Tap the display to switch the To and From units.

    World Currencies: Traveling? Use the World Currencies calculator to exchange over 150+ different Currencies world-wide. The world currencies exchange rates are updated from the Panoramic Software Inc. web site three times daily. Also includes buy and sell rates for most currencies. Simply select the currencies using the To and From buttons then enter an amount. The To currency will be instantly displayed along with the buy and sell prices.

    Smartphone Controls
    For Non-touch screen devices with standard 12 button phone keys - you'll be able to access basic calculator features using only the 12 buttons.
    For non-touch screen devices with keyboards, simply using the appropriate button for entering numbers, basic operators, etc.

    Menus All of the menus in Converter Pro have been optimized for both touch screen and non-touch screen devices. You can 'drag' the menu to scroll or use the directional pad. Press Up or Down to move the selector, or press Left and Right for rapid scrolling.

    World Currencies

    World Currencies gives you access to 150 currencies. Currencies are updated 3 times daily on the Panoramic Software Servers and can be updated any time by pressing the Update button.

    Buttons

    From - Select the currency to convert from

    Uninstalling

    To uninstall from your Windows Mobile Pocket PC: (WM Classic/WM Professional)

    • Go to Start | Settings | System tab | Remove Programs.
    • Select on the name of the Panoramic game or application and tap "Remove".
    • Follow the onscreen instructions.

    To uninstall from your Windows Mobile Smartphone: (WM Standard)

    • Go to Start | Settings | Remove Programs.
    • Select on the name of the Panoramic game or application and tap "Remove".
    • Follow the onscreen instructions.

    Your Panoramic Windows Mobile application should now have been uninstalled.

    Note, when updating your Panoramic Software, you should uninstall any old versions before installing the new version. In most cases, you can install a new trial version to update an older full version unless the new version is a paid upgrade.
