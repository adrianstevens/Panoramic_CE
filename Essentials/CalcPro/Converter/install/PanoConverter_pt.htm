    Como Usar

    Basta pressionar os botões no ecrã usando seu dedo ou stylus (WM Clássico / Profissional), ou usar o selector e teclado (WM Standard) . Use o botão de menu na parte inferior direita para mudar rapidamente para a sua calculadora desejada. O botão de ficheiro permite que você faça uso do recurso Copiar e colar, assim como contém o Sobre e opções. Dentro do Menu de Opções você pode escolher entre um número de configurações, incluindo o uso Configurações Regionais (que detecta a sua preferência região para mostrar o número - vai fazer um número longo a ser dividido com vírgulas decimais ou, se for caso disso), reproduzir sons, Gravar Fita , Calculadora Modo, Notação, Conversão Buffer, RPN tamanho de pilha,

    As Calculadoras

    Conversão Unidades: Converte 17 diferentes unidades de medida, tais como, Área, Energia, Culinária, economia de combustível, velocidade, tempo, temperatura e torque. Comece por seleccionar o tipo de conversão com o botão de conversão, as unidades seleccione com o "a" e "De" botões. Agora insira um valor e ele será imediatamente convertido. Toque no ecrã para mudar o "Para" e "De" de unidades.

    Moedas do Mundo: A viajar? Use a calculadora Mundo moedas para alternar por mais de 150 diferentes moedas do mundo inteiro. Moedas do mundo as taxas de câmbio são actualizadas no site da Panorâmica Software Inc. três vezes ao dia. Também inclui a compra e venda de tarifas para a maioria das moedas. Basta seleccionar as moedas usando o "a" e "De" botões em seguida, introduza um valor. O "Para moeda", será imediatamente apresentado junto com a compra e venda de preços.


    Controlos Smartphone
    Para dispositivos de ecrã não-touch com as 12 teclas padrão do telefone - você será capaz de aceder recursos calculadora básica usando apenas os 12 botões. Toque e segure os botões para acesso a funções secundárias. Por exemplo, mantenha pressionada o '5’ botão para se multiplicar.
    Para dispositivos de teclado apenas não-touch, basta usar o botão apropriado para digitar números, e os operadores de base, etc Para as funções avançadas, consulte a lista de atalhos de teclado.

    O Ecrâ
    Para dispositivos de ecrã de toque você pode controlar muitas das funcionalidades disponíveis nas opções tocando no ecrã. Toque o indicador "M" de memória para ver o conteúdo, isso também irá mostrar todas as dez posições de memória quando "Adv." Memória é seleccionado.
    Você também pode alterar o estado DRG e a notação tocando no ecrã. A stack RPN será também mostrado tocando na área de responder quando o modo RPN é seleccionado.

    Menus Todos os menus em Converter Pro foram optimizados para o ecrã sensível ao toque e dispositivos com ecrã não-touch. Você pode "arrastar" o menu para navegar ou usar o botão direccional. Pressione para cima ou para baixo para mover o selector ou pressione Esquerda e Direita para passar rápido.

    Conversor de Unidades

    O Conversor de Unidades da-lhe acesso a milhares de conversões em 17 categorias.

    De - A unidade que quer converter
    Para - A unidade a ser convertida
    Conversão - Escolha entre 17 tipos de conversões

    Moedas do Mundo

    Moedas do Mundo dá-lhe acesso a 150 moedas. Moedas são actualizados pelo menos 3 vezes por dia dos servidores Panoramic e pode ser actualizado a qualquer momento, premindo o botão Actualizar.

    Botões

    De - Seleccione a moda que deseja converter
    Para - Seleccione a moeda para conversão
    Actualizar - Actualize as ultimas taxas de cambio (necessita de uma ligação à internet activa)

    Utilizadores de dispositivos com ecrã sensível ao toque, podem tocar no ecrã para alternar entre as moedas.

    Desinstalação

    Para remover do Windows Mobile Pocket PC: (WM Clássico/WM Profissional)

    • Vá a Iniciar | Definições | Tab Sistema | Remover Programas.
    • Seleccione o nome do programa ou Jogo Panoramic e toque em "Remover".
    • Siga as instruções no écran.

    Para remover do Windows Mobile Smartphone: (WM Standard)

    • Vá a Iniciar | Definições | Remover Programas.
    • Seleccione o nome do programa ou Jogo Panoramic e toque em "Remover".
    • Siga as instruções no écran.

    A sua aplicação Windows Mobile deve neste momento estar removida.

    Nota, quando actualizar a sua aplicação Panoramic Software, você deve desinstalar qualquer versão antiga antes de instalar a nova versão. Na maioria dos casos, você pode instalar uma versão de teste nova para actualizar uma versão completa a menos que a nova versão é um upgrade pago.
