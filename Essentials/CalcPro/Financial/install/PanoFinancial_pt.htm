    Visão Geral

    Panoramic Financial Calc oferece mais poder, mais recursos e maior precisão do que outra calculadoras celular financiera. Com 2 calculadoras poderosas e 10 calculadoras financeiras adicionais, Contendo cinco modos de calculadora, cinqo opções de visualização, e seis temas, Panoramic Financial Calcfunciona da maneira que quiser. Voçê pode usar o Panoramic Calc Pro para realizar qualquer das operações padrão ou avançadas para que normalmente usaria uma calculadora financiera de mão.

    • Contém 10 folhas de calculadora poderosas para ajudar a resolver problemas financeiros comuns

    • Opção de respostas para o uso de configurações regionais para formato de ecrã
    • Sistema de menus fácil de usar por toque para uma navegação rápida e intuitiva
    • Opção de gravar Fita saídas de um registo de texto de todos os seus cálculos para que possa rever mais tarde
    • Seis (6) temas para escolher
    • Botões e menus pensados para funcionar através de toque com os dedos
    • Simples de usar - A maioria das opções ficam disponíveis apenas por tocar no ecrã.
    • "Input inteligente" tecnologia impede que você inserir expressões inválidas e evita erros

    Como Usar

    Basta pressionar os botões no ecrã usando seu dedo ou stylus (WM Clássico / Profissional), ou usar o selector e teclado (WM Standard) . Use o botão de menu na parte inferior direita para mudar rapidamente para a sua calculadora desejada. O botão de ficheiro permite que você faça uso do recurso Copiar e colar, assim como contém o Sobre e opções. Dentro do Menu de Opções você pode escolher entre um número de configurações, incluindo o uso Configurações Regionais (que detecta a sua preferência região para mostrar o número - vai fazer um número longo a ser dividido com vírgulas decimais ou, se for caso disso), reproduzir sons, Gravar Fita , Calculadora Modo, Notação, Conversão Buffer, RPN tamanho de pilha, Temas e a quantidade de dígitos fixos. O botão de voltar, localizado no meio da barra de menu inferior, o retornará para a calculadora utilizada pela última vez.

    As Calculadoras

    Financeira: Dez folhas poderosas permitem que você as use para resolver os mais frequentes de problemas financeiros. Escolher, juros compostos (ICOM), Data (Data), amortização (DEP), Calculador de gorjetas (Gorjeta), calculadora do empréstimo (Empréstimo), Hipoteca (Hip), Variação Percentual de Margem de Lucro (%?), (PRFT), Partida de Vendas (BrkE), e Tempo Valor Dinheiro (TVM).

    Estatísticas: A Calculadora Estatística é uma excelente ferramenta de análise para comparar percentagens, médias ou médio desempenho e gráficos de regressão linear ou cálculos de regressão exponencial.



    Controlos Smartphone
    Para dispositivos de ecrã não-touch com as 12 teclas padrão do telefone - você será capaz de aceder recursos calculadora básica usando apenas os 12 botões. Toque e segure os botões para acesso a funções secundárias. Por exemplo, mantenha pressionada o '5’ botão para se multiplicar.
    Para dispositivos de teclado apenas não-touch, basta usar o botão apropriado para digitar números, e os operadores de base, etc Para as funções avançadas, consulte a lista de atalhos de teclado.

    O Ecrâ
    Para dispositivos de ecrã de toque você pode controlar muitas das funcionalidades disponíveis nas opções tocando no ecrã. Toque o indicador "M" de memória para ver o conteúdo, isso também irá mostrar todas as dez posições de memória quando "Adv." Memória é seleccionado.
    Você também pode alterar o estado DRG e a notação tocando no ecrã. A stack RPN será também mostrado tocando na área de responder quando o modo RPN é seleccionado.

    Menus Todos os menus em Financial Calc foram optimizados para o ecrã sensível ao toque e dispositivos com ecrã não-touch. Você pode "arrastar" o menu para navegar ou usar o botão direccional. Pressione para cima ou para baixo para mover o selector ou pressione Esquerda e Direita para passar rápido.

    Opções

    Usar opções Regionais irá formatar os resultados que serão exibidos de acordo com suas configurações de dispositivo regional.
    Tocar Sons vai activar ou desactivar o som 'Toque' quando um Butão é pressionado
    Gravar Fita Vai criar um ficheiro de texto com todos os cálculos anotados. Um novo ficheiro é criado cada momento todas as vezes que Financial Calc é executado. Os ficheiros são guardados na pasta Meus Documentos.
    Memoria Avançada. permite que as características avançadas de memória. Isto irá trocar os botões de memória com botões avançados em todas as calculadoras. Utilizado em combinação com os botões numéricos você será capaz de aceder a 10 posições de memória.

    Modo Calculadora: Mudar o modo de calculadora entre Normal, Algébrica, Algébrico Directa, RPN e expressão
    Notação: Alterar a notação de saída entre Normal, Ciência, Engenharia, Fração e Fixa
    Buffer de Conversão: Mudar o tamanho do Buffer de conversão
    Tamanho de pilha RPN: Alterar o tamanho total da folha ao usar a calculadora modo RPN
    Tema: Mudar a aparência do Financial Calc
    Dígitos Fixos: Mudar o numero de casas decimais apresentados na folha.

    Calculadora Financeira

    A Calculadora Financeira dá-lhe acesso a 10 folhas poderosas para o cálculo quase todos os problemas financeiros comuns.

    Para usar as folhas de trabalho basta digitar os valores para os campos desejados e pressione o botão "Calc no campo (s) que você deseja calcular.

    Se um valor é atenuado ou cinza, isso significa que só pode ser resolvido, mas não entrou.

    Além disso, algumas folhas são projectados para resolver apenas os valores específicos, nestes casos, o selector de ecrã saltará para o valor calculado quando o botão "Calc é pressionado.

    As folhas de trabalho usam o setor mais preciso equações padronizadas para o cálculo dos seus resultados. Descrição dos cálculos está além do escopo do documento de ajuda.

    Butões

    ICom - Abra a écran de juros compostos
    Date - Abra a folha de trabalho de data financeira
    Dep - Abra a folha de amortização
    Gorjeta - Abra a folha de grojeta
    Empréstimo - Abra a folha de empréstimo
    Amor - Abra a folha de amortização
    %Delta - Folha de mudança de percentagem
    Prft - Folha de margem de lucro
    BrkE - Folha de empréstimo
    TVM - TFolha de tempo e valor

    Por favor consulte a calculadora cientifica para ver a descrição dos outros botões.

    Calculador de Estatisticas

    A calculadora Estatísticas lhe dá acesso a funções comuns de estatística. Por favor, veja a calculadora financeira para obter informações sobre as folhas de trabalho.

    Butões

    nPr - O Numero de permutações
    nCr - O numero de combinações possíveis
    LinReg - Folha de regressão linear ( f(x) = A+Bx )
    ExpReg - Folha de regressão exponencial ( f(x) = A*B^x )

    A Regressão Linear de folha de trabalho (LinReg) vai tentar o seu modelo de dados para uma linha reta na forma f(x) = A+B*x, a folha de trabalho Exponencial (ExpReg) servirá de modelo para uma curva exponencial na forma de f(x) = A*B^x.
    Ambas as folhas de trabalho: Média e desvio de ambos os X & Y, a soma de X, Y, X^2, Y^2 and X*Y.
    A Regressão Linear da folha de trabalho irá calcular o intercepto Y, Declive, Regressão e Determinação da melhor equação.
    A folha de trabalho de regressão exponencial também vai calcular A & B para a equação de melhor ajuste, bem como a regressão e determinação.

    Como inserir dados e ver resultados:
    Depois de uma folha de trabalho estatística ter sido aberta, você pode começar a inserir dados. X (1) representa o valor de X no primeiro par de dados, Y (1) representa o valor de y em primeiro lugar. Mova o selector para escolher um local e inserir valores usando os botões no écran. Definir valores para os pares que desejar e pressione o botão "Calc" para mostrar os resultados estatísticos. Você pode gerar um gráfico dos dados e da equação ajustada pressionando o Butão 'gráfico'. Os dados podem ser Gravados ou carregados de / para arquivos CSV usando os botões Carregar 'e' Gravar'.

    Consulte a calculadora científica para as descrições dos outros botões.

    Desinstalação

    Para remover do Windows Mobile Pocket PC: (WM Clássico/WM Profissional)

    • Vá a Iniciar | Definições | Tab Sistema | Remover Programas.
    • Seleccione o nome do programa ou Jogo Panoramic e toque em "Remover".
    • Siga as instruções no écran.

    Para remover do Windows Mobile Smartphone: (WM Standard)

    • Vá a Iniciar | Definições | Remover Programas.
    • Seleccione o nome do programa ou Jogo Panoramic e toque em "Remover".
    • Siga as instruções no écran.

    A sua aplicação Windows Mobile deve neste momento estar removida.

    Nota, quando actualizar a sua aplicação Panoramic Software, você deve desinstalar qualquer versão antiga antes de instalar a nova versão. Na maioria dos casos, você pode instalar uma versão de teste nova para actualizar uma versão completa a menos que a nova versão é um upgrade pago.
