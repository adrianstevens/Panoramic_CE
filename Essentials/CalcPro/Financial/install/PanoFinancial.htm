    How To Use

    You can use Panoramic Financial Calc to perform simple, financial or statistical calculations.  Simply press the onscreen buttons using your finger or stylus (WM Classic/Professional), or use the selector and keypad (WM Standard). Use the menu button at the bottom right to quickly switch to your desired calculator. The file button allows you to make use of the Copy & Paste feature as well as contains the About and Options screens. Within the Options Menu you can choose between a number of settings, including Use Regional Settings (which detects your region's preference for number display - it will cause long numbers to be broken up with commas or decimals, where appropriate), Play Sounds, Record Tape, Calculator Mode, Notation, Conversion Buffer, RPN Stack Size, Custom Skins and the amount of Fixed Digits. The toggle button, located in the middle of the bottom menu bar, will take you back to the last calculator used.

    Financial: Ten powerful worksheets allow you to solve the most frequent of financial problems. Choose from, Compound Interest (ICom), Date (Date), Depreciation (Dep), Tip Calculator (Tip), Loan Calculator (Loan), Mortgage (Mort), Percent Change (%?), Profit Margin (Prft), Break Even Sales ( BrkE), and Time Value Money (TVM).

    Statistics: The Statistics Calculator is an outstanding analyzing tool for comparing percentages, mean averages or performing and graphing linear regression or exponential regression calculations.

    Smartphone Controls
    For Non-touch screen devices with standard 12 button phone keys - you'll be able to access basic calculator features using only the 12 buttons. Tap and hold buttons to access the secondary functions. For example, hold down the '5' key to multiply.
    For non-touch screen devices with keyboards, simply using the appropriate button for entering numbers, basic operators, etc. For advanced functions, please refer to the keyboard shortcut list.

    The Display
    For Touch screen devices you can control many of the features available in the options by tapping on the display. Tap the 'M' memory indicator to see the contents, this will also show all ten memory locations when "Adv. Memory" is selected.
    You can also change the DRG state and the notation by tapping on the display. The RPN stack will also be shown by tapping on the answer area when RPN mode is selected.

    Menus All of the menus in Finanical Calc have been optimized for both touch screen and non-touch screen devices. You can 'drag' the menu to scroll or use the directional pad. Press Up or Down to move the selector, or press Left and Right for rapid scrolling.

    Options

    Use Regional Settings will format the display results according to your devices regional settings.
    Play Sounds will enable or disable the 'tap' sound when a button is pressed
    Record Tape will create an plain text file will all calculations noted. A new file is created every time Fiancial Calc is run. The files are stored in the My Documents Folder.
    Adv. Memory enables the advanced memory features. This will swap the memory buttons with advanced buttons on all calculators. Used in combination with the number buttons you'll be able to access 10 memory locations.
    Calculator Mode: Change the calculator mode between Normal, Algebraic, Direct Algebraic, R.P.N and expression
    Notation: Change the output notation between Normal, Scientific, Engineering, Fraction and Fixed
    Conversion Buffer: Change the size of the buffer used for base conversions
    RPN Stack Size: Change the total size of the stack when using the RPN calculator mode
    Skin: Change the appearance of Fiancial Calc
    Fixed Digits: Change the number of decimal places displayed when using the Fixed notation.

    Financial Calculator

    The Financial Calculator gives you access to 10 powerful worksheets for calculating almost all common financial problems.
    To use the worksheets simply enter the values for your desired fields and press the "Calc" button on the field(s) you wish to calculate.
    If a value is lightened or greyed it means it can only be solved but not entered.
    Additionally, some worksheets are designed to only solve specific values, in these cases, the onscreen selector will jump to calculated value when the "Calc" button is pressed.

    The worksheets all use the most accurate industry standardized equations for calculating their results. Description of the calculations is beyond the scope of the help document.

    Buttons

    ICom - Open the compound interest worksheet
    Date - Open the Financial date worksheet
    Dep - Open the depreciation worksheet
    Tip - Open the Tip worksheet
    Loan - Open the loan worksheet
    Mort - Open the Mortgage worksheet
    %Delta - Percentage Change worksheet
    Prft - Profit Margin worksheet
    BrkE - Break Even worksheet
    TVM - Time Value Money worksheet

    Please see the Scientific Calculator for descriptions of the other buttons.

    Uninstalling

    To uninstall from your Windows Mobile Pocket PC: (WM Classic/WM Professional)

    • Go to Start | Settings | System tab | Remove Programs.
    • Select on the name of the Panoramic game or application and tap "Remove".
    • Follow the onscreen instructions.

    To uninstall from your Windows Mobile Smartphone: (WM Standard)

    • Go to Start | Settings | Remove Programs.
    • Select on the name of the Panoramic game or application and tap "Remove".
    • Follow the onscreen instructions.

    Your Panoramic Windows Mobile application should now have been uninstalled.

    Note, when updating your Panoramic Software, you should uninstall any old versions before installing the new version. In most cases, you can install a new trial version to update an older full version unless the new version is a paid upgrade.
