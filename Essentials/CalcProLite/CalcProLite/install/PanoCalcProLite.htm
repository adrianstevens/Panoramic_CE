    How To Use

    You can use Panoramic Calc Pro Lite to perform any of the standard operations for which you would normally use a handheld calculator.  Simply press the onscreen buttons using your finger or stylus (WM Classic/Professional), or use the selector and keypad (WM Standard). Use the menu button at the bottom right to quickly switch to your desired calculator. The file button allows you to make use of the Copy & Paste feature as well as contains the About and Options screens. Within the Options Menu you can choose between a number of settings, including Use Regional Settings (which detects your region's preference for number display - it will cause long numbers to be broken up with commas or decimals, where appropriate), Play Sounds, Record Tape, Calculator Mode, Notation, Conversion Buffer, RPN Stack Size, Custom Skins and the amount of Fixed Digits. The toggle button, located in the middle of the bottom menu bar, will take you back to the last calculator used.

    Standard: Useful for most calculations in everyday use. Includes functions such as exponent, square root, percentage, inverse, backspace and memory functions. The simple layout and large buttons make it easy to use with your fingers.

    European Currencies: Traveling? European Currencies allows you to check the currency rates of 13 different countries and compare them. To check over 150 different world currencies please upgrade to Panoramic Calc Pro full edition.

    Smartphone Controls
    For Non-touch screen devices with standard 12 button phone keys - you'll be able to access basic calculator features using only the 12 buttons. Tap and hold buttons to access the secondary functions. For example, hold down the '5' key to multiply.
    For non-touch screen devices with keyboards, simply using the appropriate button for entering numbers, basic operators, etc. For advanced functions, please refer to the keyboard shortcut list.

    The Display
    For Touch screen devices you can control many of the features avaliable in the options by tapping on the display. Tap the 'M' memory indicator to see the contents, this will also show all ten memory locations when "Adv. Memory" is selected.
    You can also change the DRG state and the notation by tapping on the display. The RPN stack will also be shown by tapping on the answer area when RPN mode is selected.

    Menus All of the menus in Calc Pro have been optomized for both touch screen and non-touch screen devices. You can 'drag' the menu to scroll or use the directional pad. Press Up or Down to move the selector, or press Left and Right for rapid scrolling.

    Options

    Use Regional Settings will format the display results according to your devices regional settings.
    Play Sounds will enable or disable the 'tap' sound when a button is pressed
    Record Tape will create an plain text file will all calculations noted. A new file is created every time Calc Pro Lite is run. The files are stored in the My Documents Folder.
    Adv. Memory enables the advanced memory features. This will swap the memory buttons with advanced buttons on all calculators. Used in combination with the number buttons you'll be able to access 10 memory locations.
    Calculator Mode: Change the calculator mode between Normal, Algebraic, Direct Algebraic, R.P.N and expression
    Notation: Change the output notation between Normal, Scientific, Engineering, Fraction and Fixed
    Conversion Buffer: Change the size of the buffer used for base conversions
    RPN Stack Size: Change the total size of the stack when using the RPN calculator mode
    Skin: Change the appearance of Calc Pro Lite
    Fixed Digits: Change the number of decimal places displayed when using the Fixed notation.

    Uninstalling

    To uninstall from your Windows Mobile Pocket PC: (WM Classic/WM Professional)

    • Go to Start | Settings | System tab | Remove Programs.
    • Select on the name of the Panoramic game or application and tap "Remove".
    • Follow the onscreen instructions.

    To uninstall from your Windows Mobile Smartphone: (WM Standard)

    • Go to Start | Settings | Remove Programs.
    • Select on the name of the Panoramic game or application and tap "Remove".
    • Follow the onscreen instructions.

    Your Panoramic Windows Mobile application should now have been uninstalled.

    Note, when updating your Panoramic Software, you should uninstall any old versions before installing the new version. In most cases, you can install a new trial version to update an older full version unless the new version is a paid upgrade.
