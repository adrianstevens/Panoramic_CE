Présentation

LaunchPad permet la liberté d'accéder facilement à vos applications Windows Mobile, les paramètres, fichiers et dossiers avec vos doigts. Flip entre deux écrans personnalisables favoris pour accéder rapidement à vos questions les plus fréquemment utilisés. Un contact facile à utiliser cinétique menu déroulant vous permet de retrouver le programme ou le périphérique Configuration vous cherchez en quelques secondes.

Retour

Comment utiliser

Pour naviguer entre les favoris, les programmes et les paramètres, soit appuyer sur le bouton de l'écran désiré ou faites glisser le menu de l'onglet noir. En tapant sur le menu, sélectionnez l'onglet (noir) à n'importe quel moment apparaît le menu principal permettant d'accéder aux options, l'aide doc, ou sur écran.

Favoris écran

L'écran Favoris s'affiche en premier. Pour ajouter un programme ou un réglage pour cet écran, il suffit de sélectionner le "+" icône / bouton dans un emplacement vide. Une cinétique menu apparaîtra vous permettant de choisir et d'insérer le programme désiré ou de réglage. Vous pouvez vous déplacer d'un programme / réglage / fichier / dossier à n'importe quel emplacement vide en sélectionnant et en maintenant enfoncé sur l'icône et en le faisant glisser à l'emplacement désiré avec le doigt. Exploiter une icône de lancement du programme / réglage / fichier / dossier. Pour supprimer un programme / réglage / fichier / dossier tout simplement appuyer sur l'onglet favoris de révéler un menu déroulant et choisissez Supprimer. Basculer rapidement entre les écrans de favoris à l'aide du curseur en haut de l'écran.

Programmes de l'écran

Par défaut, l'écran affiche les programmes de votre appareil des programmes de A à Z, la liste de tous les dossiers en haut de l'écran. Faites défiler la liste des programmes cinétique rapidement en utilisant votre doigt ou le stylet. En tapant ou en appuyant sur le d-pad sur un programme sélectionné, vous lancer ce programme.

Paramètres de l'écran

Par défaut, l'écran affiche les paramètres de votre appareil personnel, des réglages, et les connexions de A à Z. Faites défiler la liste rapidement cinétique à l'aide de votre doigt ou le stylet. En tapant ou en appuyant sur le d-pad sur un paramètre que vous lance l'écran.

Options

Accéder au menu d'options simplement en tapant la barre d'outils, sélectionnez l'onglet pour afficher une pop up menu. Choisissez les options.

Général, de couleurs: vous permet de changer la couleur de peau.

Général, le menu Démarrer: sélectionnez "Sur" ou "Hors tension" pour avoir le menu démarrer LaunchPad lancer le programme.

Général, aujourd'hui, l'écran du menu Set: choisir "Sur" ou "Hors tension" pour mettre LaunchPad au bas de votre écran aujourd'hui.

Général, Set icône du menu démarrer: choisir entre le "Défaut", "Windows", ou "Fusée" de changer l'icône affichée en haut à gauche le menu démarrer.

Favoris, Icon taille: choisir entre le "Grand", “Ordinaire", ou "Petit" pour changer la taille des icônes sur l'écran Favoris.

Favoris, Gauche curseur titre: choisir entre "Programmes", "Jeux", "Apps", "Paramètres", "Favoris" ou "Utilitaires " pour la gauche, la barre de titre.

Favoris, Droit curseur titre: choisir entre "Programmes", "Jeux", "Apps", "Paramètres", "Favoris" ou “Utilitaires " pour le droit de la barre de titre.

Favoris, Afficher le nom: choisir "Sur" ou "Hors tension" pour afficher le nom du programme ou les paramètres sur l'icône (favoris écran).

Désinstallation

Pour désinstaller de votre Pocket PC Windows Mobile (WM Classic / WM Professional)

  • Allez dans Démarrer | Paramètres | onglet Système | Suppression de programmes.
  • Sélectionnez le nom du jeu ou une application panoramique et appuyez sur "Supprimer".
  • Suivez les instructions à l'écran.

Votre Panoramic Windows Mobile application devrait maintenant avoir été désinstallé.

Notez que lorsque vous mettre à jour vos logiciels panoramique, vous devez désinstaller toutes les anciennes versions avant d'installer la nouvelle version. Dans la plupart des cas, vous pouvez installer une nouvelle version d'essai pour mettre à jour une ancienne version complète à moins que la nouvelle version est une mise à niveau payés.
