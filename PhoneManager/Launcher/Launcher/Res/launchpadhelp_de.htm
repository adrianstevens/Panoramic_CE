Überblick

Panoramic LaunchPad ermöglicht Ihnen den fingeroptimierten Zugriff auf Ihr Windows Mobile-Anwendungen, Einstellungen, Dateien und Ordner. Wechseln Sie zwischen zwei konfigurierbaren Favoritenansichten für den schnellen Zugriff auf Ihre am häufigsten verwendeten Anwendungen. Mit einer komfortabel zu bedienenden kinetischen Scroll-Liste finden Sie das Programm oder die Systemeinstellung, die Sie suchen, in Sekunden.

Zurück

Verwendung

Um zwischen der Favoriten-, Programme- und Einstellungen-Ansicht zu navigieren, tippen Sie entweder auf die jeweilige Registerkarte oder ziehen den schwarzen Auswahlbalken auf den gewünschten Tab. Zugriff auf die Optionen, die Hilfe und den Infoschirm erhalten Sie, wenn Sie bei selektierter Programme-Registerkarte die Registerkarte erneut anwählen.

Favoriten-Ansicht

Direkt nach dem Programmstart wird Ihnen die Favoriten-Liste angezeigt. Um ein Programm oder eine Systemeinstellung zur aktuellen Ansicht hinzuzufügen, müssen Sie lediglich das “+”-Symbol auf einem leeren Eintrag antippen. Daraufhin erscheint eine kinetische Scroll-Liste mittels der Sie das gewünschte Programm oder die Systemeinstellung auswählen können. Ein bereits belegter Eintrag lässt sich innerhalb der Ansicht verschieben, indem Sie einen bestehenden Eintrag gedrückt halten und das Symbol dann mit dem Finger auf die neue Position ziehen. Um das Programm, die Systemeinstellung, die Datei oder den Ordner aufzurufen, tippen Sie einfach auf das Symbol des gewünschten Eintrags. Um einen Eintrag zu entfernen tippen Sie einfach auf die Favoriten-Registerkarte, um im darauf erscheinenden Popup-Menü den Menüpunkt  "Entfernen" auszuwählen. Mit dem Schieberegler am oberen Rand des Bildschirms können Sie schnell zwischen den zwei konfigurierten Favoriten-Bildschirmen umschalten.

Programme-Ansicht

In der In der Programme-Ansicht werden standardmässig alle Anwendungen aus dem Startmenü Ihres  Gerätes von A bis Z aufgelistet (Ordner am Anfang der Liste). Blättern Sie komfortabel per Finger oder Stylus durch die kinetische Scroll-Liste. Um ein Programm aus der Liste zu starten, müssen Sie es lediglich antippen oder die Auswahl-Taste des D-Pads betätigen.

Einstellungen-Ansicht

In der Einstellungen-Ansicht werden alle Optionen aus den Systemmenüs “Persönliche Einstellungen”, “Einstellungen” und “Verbindungen” Ihres Gerätes von A bis Z aufgelistet. Blättern Sie komfortabel per Finger oder Stylus durch die kinetische Scroll-Liste. Um eine Option  aus der Liste zu starten, müssen Sie es lediglich antippen oder die Auswahl-Taste des D-Pads betätigen.

Optionen

Die Registerkarten bieten ein Options-Menue, das sich öffnen lässt, in dem man die aktuell gewählte Registerkarte ein weiteres Mal antippt.

Allgemein, Farbschema: Mit dieser Option können Sie die Farbe des Skins verändern.

Allgemein, Startmenü: Wählen Sie "Aus" oder "Ein", um das Windows Mobile Startmenü mit LaunchPad zu ersetzen.

Allgemein, Heute-Bildschirm Softkey: Wählen Sie "Aus" oder "Ein", um LaunchPad über einen Softkey auf dem Heute-Bildschirm starten zu können.

Allgemein, Startmenü-Symbol: Wählen Sie zwischen "Standard", "Windows" oder "Rocket" als Ersatz für das Windows Mobile Startmenü-Symbol.

Favoriten, Symbolgröße: Wählen Sie zwischen "Groß", "Normal" oder "Klein" als Grösse der Symbole für die Favoriten-Ansicht.

Favoriten, Titel linker Schieberegler: Wählen Sie zwischen "Programme", "Spiele", "Anwendungen", "Einstellungen", "Favoriten" oder "Hilfsprogramme" als Titel für den linken Schieberegler.

Favoriten, Titel rechter Schieberegler: Wählen Sie zwischen "Programme", "Spiele", "Anwendungen", "Einstellungen", "Favoriten" oder "Hilfsprogramme" als Titel für den rechten  Schieberegler.

Favoriten, Namen anzeigen: Wählen Sie "Aus" oder "Ein", um den Namen des Programmes oder der Einstellung zusätzlich zum Symbol in der Favoriten-Ansicht anzuzeigen.

Deinstallation

So deinstallieren Sie die Anwendung von Ihrem Windows Mobile Pocket PC: (WM Classic / WM Professional)

  • Gehen Sie auf Start -> Einstellungen -> Registerkarte System -> Programme entfernen
  • Wählen Sie den Namen der Panoramic-Anwendung, die Sie  deinstallieren möchten und tippen Sie auf "Entfernen".
  • Folgen Sie den Anweisungen auf dem Bildschirm.

Ihr Panoramic Windows Mobile-Anwendung sollte nun deinstalliert werden.
Bitte beachten Sie, dass Sie vor der Installation der neuen Version einer Panoramic-Anwendung zuerst die alte Version auf Ihrem Gerät deinstallieren sollten. In den meisten Fällen reicht die Installation einer neuen Testversion, um ein Update für die bereits installierte Vollversion durchzuführen (sofern es sich nicht um ein bezahltes Update handelt).
