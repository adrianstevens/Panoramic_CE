Resumo

Panoramic LaunchPad permite aceder às suas aplicações Windows Mobile, configurações, contactos, arquivos e pastas com a ponta dos seus dedos. Deslise entre vários ecrãns personalizaveis, favoritos para acesso rápido aos seus items mais utilizados. Com um menu facil de usar com movimento cinético permite-lhe encontrar o programa ou dispositivo que procura em segundos.

  • Use até seis (6) ecrãns favoritos costumizaveis
  • Arraste, largue e lance programas, configurações, contatos, arquivos e patas nos ecrans favoritos
  • Navegação facil, menu para ser usado com os dedos, e movimento cineticos de listas
  • Seis (6) cores de realçe personalizadas
  • Personalize o seu fundo, escolhendo uma imagem, usando o seu tema de telefone atual, ou tirar uma foto com a câmera do dispositivo

Como Usar

Para navegar entre os favoritos, programas e configurações, faça qualquer toque no botão de ecrã desejado, ou arraste a guia do menu. Tocando na guia de menu selecionado a qualquer momento irá aparecer no menu principal que permite acesso às opções, ajuda, ou acerca dos ecrãns.

Ecrã de Favoritos

O ecrã de Favoritos aparecerá quando você abrir o Launch Pad. Para adicionar um atalho para este ecrã simplesmente selecione o Icone/botão "+" em um slot vazio. Um menu pop-up irá aparecer permitindo que escolha e insira o item desejado. Pode também mover um programa/configuração/contacto / Ficheiro/Pasta para qualquer slot vazio, selecionando e mantendo pressionada sobre o ícone (por um segundo) e arrastando-o para o local desejado com o dedo. Tocando num ícone irá lançar o atalho desejado. Para remover um atalho simplesmente toque e segure o ícone (por um segundo) e arrastá-lo em cima do ícone da Lixeira (em baixo à direita). Alterne rapidamente entre os favoritos e ecrãns deslizando esquerda ou direita no ecrã.

Ecrã Programas

O ecrã programas mostrará os programas do dispositivo de A a Z, listando todas as pastas na parte superior do ecrã. Percorra a lista de programas cinética rapidamente usando o dedo ou caneta. Ao tocar ou pressionar o D-pad num programa selecionado irá lançar o programa.

Ecrã de Definições

Por padrão, o ecrã Definições irá mostrar no seu dispositivo, configurações e ligações a partir de A a Z. Deslize pela lista cinética rapidamente usando o dedo ou caneta. Ao tocar ou pressionar o D-pad em uma configuração que você selecionou irá lançar nesta tela.

Opções

Aceda ao menu de opções com um simples toque no separador seleccionado ferramentas para revelar um menu pop-up. Escolha as opções.

Geral - Esquema de cores: permite alterar a cor do tema.

Geral - Fundo: permite que você selecione o fundo desejado.

Geral - Uso em Ecrã Inteiro: Definir Launchpad como um aplicativo de ecrã inteiro ou não.

Geral - Menu Iniciar: escolha a opção "Ligar" ou "Desligado" para ter o menu iniciar o lançamento do programa Launchpad.

Geral - Menu de ecrã hoje: escolher "Ligar" ou "Desligar" para definir LaunchPad na parte inferior do ecrã de hoje.

Geral - Ícone de menu iniciar definir: escolher entre o "Padrão", "Windows", ou "Foguete" para alterar o ícone exibido no menu iniciar superior esquerdo.

Favoritos - Tamanho de Icones: escolher entre "grandes", "Normal", ou "pequeno" para alterar o tamanho dos ícones no ecrã Favoritos.

Favoritos - Desenhar Telhas: desenhar as telhas de fundo no ecrã Favoritos.

Favoritos - Mostrar nome: escolha a opção "Ligado" ou "Desligado" para exibir nomes de atalhos no ecrã Favoritos.

Desinstalação

Para desinstalar do seu Windows Mobile Pocket PC: (WM Classic/WM Professional)

  • Vá a Iniciar | Definições | Sistema | Remover Programas.
  • Seleccionar no nome do jogo Panoramic ou aplicação e escolher "Remover".
  • Siga as instruções no ecrã.

A Sua aplicação Panoramic Windows Mobile deverá então ser removida.

Note, quando actualizar o seu Software Panoramic, deve desinstalar as versões antigas antes de instalar uma nova versão. Na maioria dos casos, pode instalar uma nova versão de experimental para actualizar uma versão anterior ou para actualizar uma versão completa mais antiga, a não ser que a nova versão seja uma actualização paga.
