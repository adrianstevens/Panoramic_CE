Overview

Wondering who just called you? Discover where calls originated from with Panoramic Area Code Genius. Search for any North American Area Code, Australian, New Zealand, United Kingdom or world Country Code and discover its location and simple facts about that region. Alternatively, use the finger friendly touch list to search for area or country codes alphabetically by location.

Features
  • Easy to use touch buttons & kinetic scrolling list
  • Search by World Country Code, North American(NANP), Australia, New Zealand or United Kingdom Area Codes
  • Discover facts on world regions such as population and currencies

How To Use

After launching the app simply choose either "Area Codes" or "Country Codes" in the Search by box. By choosing "Area Codes" you will then be able to enter any area code located within the North American Area Codes (NANP) and discover its location and simple facts on the region. By choosing "Country Codes" you can enter any world country code to show its location and simple facts on that country.

After entering a number, press "Locate" to discover its location in the world. Please note if you have entered an unused number "unknown" will appear in the number box. Press "Clear" to start again.

Alternatively, you can press the "List" menu button to display a complete list of all country codes or area codes depending on the specified mode. Select a Country or Region from the list to display more information.

Uninstalling

To uninstall from your Windows Mobile Pocket PC: (WM Classic/WM Professional)

  • Go to Start | Settings | System tab | Remove Programs.
  • Select on the name of the Panoramic game or application and tap "Remove".
  • Follow the onscreen instructions.

Your Panoramic Windows Mobile application should now have been uninstalled.

Note, when updating your Panoramic Software, you should uninstall any old versions before installing the new version. In most cases, you can install a new trial version to update an older full version unless the new version is a paid upgrade.
