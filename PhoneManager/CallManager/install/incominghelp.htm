Overview

Never wonder where someone is calling from again. Call Genius displays the location of the number calling you and allows you to set custom calling rules or blocking annoying callers. A Today Screen Plugin allows you to set your phone to different profiles quickly. (*Note: Panoramic Call Genius requires call display to work properly).

How To Use

Call Genius

Today Screen

To display the Today Screen Plugin, choose "Start" on your Windows Mobile phone. Go to "Settings" and then choose the "Today" icon. Choose "Items" and make sure "Phone Genius" is checked. Uncheck "Phone Genius" to remove the plugin from your phone's today screen. Once installed you can use the today plugin to easily switch between different phone profiles such as, Normal, Vibrate, Silent, Silent for 3 hours or Airplane. The Today plugin will also notify of any emails, sms messages, voicemail or missed calls. Tap and hold down over the today screen application to display a menu giving you the option to add/remove/lock/arrange items.

Incoming Calls

Upon receiving an incoming call the screen will display a 3D globe that will stop at the caller area/country code location and display the area and or country. If the contact is within your contacts list, the image and name of the caller will appear. You can choose to "Ignore" or "Answer" the call. Once in a call the Cheek Guard will automatically start up to protect your call. This feature can be removed in the "Call Options". You can press "up", "down" or the center button of your d-pad to mute an incoming call without having it go straight to voicemail.

Post Call Screen

This screen displays the call information. Caller, name, number, duration and a variety of options. From this screen you can choose to block a caller using the slider bar along the bottom. You can send a custom SMS Response, add the caller to your contacts list or simply return the call. The Blocked Caller list can be managed from within the "Call Options". "More Info" button will display call information and about the area/country the call originated from.

Phone Options

The "Phone Options" icon can be found within the "Phone Genius" folder of your program files. Phone Options allows you to set the "General Options", "Caller Block List", "Calling Rules" and "Ringtones".

General Options:
Show incoming call screen: Yes or No
Show post call screen: Yes or No
Show cheek guard: Yes or No
Phone profile: Choose between Normal, Airplane, Vibrate, Silent, Silent for 3 hours

Caller Block List:
Choose from, New, Contact or Remove.

Calling Rules: Add or Remove
Ignore calls during appointment with busy status
Never allow calls from contact... (choose from contacts list)
Never allow calls from category... (choose between "business" or "personal")
Always allow calls from contact... (choose from contacts list)
Always allow calls from category... (choose between "business" or "personal")
Turn ringer off during time... (choose "start" and "end" time)

Ringtones:
Add, Edit or Remove ringtones.

Uninstalling

To uninstall from your Windows Mobile Pocket PC: (WM Classic/WM Professional)

  • Go to Start | Settings | System tab | Remove Programs.
  • Select on the name of the Panoramic game or application and tap "Remove".
  • Follow the onscreen instructions.

Your Panoramic Windows Mobile application should now have been uninstalled.

Note, when updating your Panoramic Software, you should uninstall any old versions before installing the new version. In most cases, you can install a new trial version to update an older full version unless the new version is a paid upgrade.
