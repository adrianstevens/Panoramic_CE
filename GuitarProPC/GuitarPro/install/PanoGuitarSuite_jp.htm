概要

ギタープレーヤーに必携のアプリケーション。パノラミックギタースイートには次の5つのスクリーンがあり、(コード/スケール/デジタルチューナー/耳チューング/メトロノーム)ギタリストにとって大切なことをすべてサポートします。　ギタースイートのライブラリには良く使われている25,000以上のコードがあり、練習や参考、そして教える時使えます。ビジュアル表示もあるとても正確なメトロノームは様々なテンポと拍子を刻み、練習中にテンポを一定に保つ役に立ちます。

  • 25,000以上のコードと50のスケールがあるライブラリーと、とても正確なデジタルチューナー。
  • 19のギター用(ベースギターを含む)と　4つのリュート用のチューニング
  • とても正確なメトロノーム(テンポと拍子は可変設定可能)
  • 演奏モードでは指先で弦またはフレットをタップして簡単なメロディーを演奏できます。
  • 演奏モードではすべてのスケールを表示でき様々なチューニングに対応します。
  • 右利き/左利きの切り替え可能。スキンは7種類あります。

アンインストールする

Windows Mobile Pocket PC: (WM Classic/WM Professional) からアンインストールするには

  • [スタート] → [設定] → [システム]タブ → [プログラムの削除] を開きます。
  • Panoramic game か application を選択して "削除" をタップしてください。
  • あとは画面上の指示に従ってアンインストールください。

これであなたの Panoramic Windows Mobile application はアンインストールされます。

注） Panoramic Software を更新する際は、新しいバージョンをインストールする前に古いバージョンを削除する必要があります。ほとんどの場合、新しいバージョンが有償のアップグレード限定でない限り、古い完全版を更新するために新しいお試し版をインストールすることが出来ます。
