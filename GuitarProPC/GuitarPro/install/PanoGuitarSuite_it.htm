Panoramica

Panoramic Guitar Suite è una raccolta di accordi per la chitarra da utilizzare su Windows Mobile Phone. Sono incluse 5 applicazioni che facilitano la gestione della chitarra.

  • Accesso ad un database di oltre 25.000 accordi per chitarra e 19 impostazioni
  • Sintonizzatore digitale
  • Metronomo
  • 50 Basculle
  • Destra e sinistra consegnato l'opzione di layout
  • Sette (7) diversi temi

Disinstallazione

Per disinstallare dal proprio Windows Mobile Pocket PC: (WM Classic / WM Professional)

  • Andare su Start | Impostazioni | scheda Sistema | Installazione applicazioni.
  • Scegli il nome del gioco panoramica o l'applicazione e toccare "Rimuovi".
  • Seguire le istruzioni sullo schermo.

Vista la tua applicazione Windows Mobile dovrebbe ora sono stati disinstallati.

Notate che, quando l'aggiornamento del software Vista, è necessario disinstallare eventuali versioni precedenti prima di installare la nuova versione. Nella maggior parte dei casi, è possibile installare una nuova versione di prova per aggiornare una versione precedente completa a meno che la nuova versione è un aggiornamento pagato.
